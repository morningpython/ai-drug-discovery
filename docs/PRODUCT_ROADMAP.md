- ✅ 데이터 파이프라인 v1.0

#### Month 2: 핵심 AI 모델 개발 (Part 1)

**Week 1-2: 분자 표현 학습 (Molecular Representation)**
- [ ] 그래프 신경망 (GNN) 구현
  - Message Passing Neural Network (MPNN)
  - Graph Attention Network (GAT)
  - 분자를 그래프로 표현 (원자=노드, 결합=엣지)
- [ ] 사전학습 모델 파인튜닝
  - ChemBERTa (SMILES 기반 Transformer)
  - MolGPT (분자 생성용)
- [ ] 벤치마크 테스트
  - MoleculeNet 데이터셋으로 검증
  - 목표 정확도: 85%+ (분자 특성 예측)

**Week 3-4: ADMET 예측 모델**
- [ ] Multi-task Learning 모델 구축
  - **A**bsorption: 흡수율 예측
  - **D**istribution: 체내 분포 예측
  - **M**etabolism: 대사 경로 예측
  - **E**xcretion: 배설 속도 예측
  - **T**oxicity: 독성 예측 (hERG, 간독성 등)
- [ ] 훈련 데이터셋 구축
  - 공개 ADMET 데이터 (10,000+ 화합물)
  - 문헌에서 추출한 실험 데이터
- [ ] 모델 검증
  - Cross-validation (5-fold)
  - 목표: AUROC 0.85+

**주요 Deliverable**:
- ✅ GNN 분자 인코더 v1.0
- ✅ ADMET 예측 API (정확도 85%+)
- ✅ 모델 성능 보고서

#### Month 3: 분자 생성 AI 개발

**Week 1-2: 생성 모델 구현**
- [ ] VAE (Variational Autoencoder) 기반 생성
  - Latent space에서 분자 샘플링
  - SMILES 디코딩
- [ ] Transformer 기반 생성
  - MolGPT/ChemGPT 구조
  - Beam search로 다양한 후보 생성
- [ ] 생성 필터링
  - Lipinski's Rule of 5 체크
  - 합성 가능성 평가 (SAScore)
  - 신규성 체크 (기존 DB와 비교)

**Week 3-4: 타겟 기반 생성**
- [ ] 조건부 생성 (Conditional Generation)
  - 원하는 특성을 조건으로 제공
  - 예: "간독성 낮음 + GLP-1R 친화도 높음"
  - 예: "JAK1/2 선택적 억제 + 두피 투과성 높음 (탈모 치료)"
- [ ] 질환별 생성 모델 특화
  - 간염 모델: HBV 억제 화합물 생성
  - GLP-1 모델: 대사 안정성 강화
  - 알츠하이머 모델: BBB 투과성 고려
- [ ] 생성 속도 최적화
  - GPU 가속 (10,000개/시간 목표)
  - 배치 생성 파이프라인

**주요 Deliverable**:
- ✅ 분자 생성 엔진 v1.0
- ✅ 질환별 특화 모델 3개
- ✅ 생성 API 및 웹 인터페이스

### Q2 (4-6월): 최적화 및 첫 번째 후보 물질 도출

#### Month 4: 강화학습 기반 분자 최적화

**Week 1-2: RL 환경 구축**
- [ ] Reward function 설계
  - ADMET 점수 (40%)
  - 타겟 친화도 예측 (30%)
  - 합성 가능성 (20%)
  - 신규성 (10%)
- [ ] PPO (Proximal Policy Optimization) 구현
  - Agent: 분자 구조 수정 액션
  - State: 현재 분자의 임베딩
  - Reward: 위 함수 기반

**Week 3-4: 반복 최적화**
- [ ] Lead optimization 파이프라인
  - 초기 분자에서 시작
  - 100-1000회 반복으로 개선
  - 다양성 유지 (diversity penalty)
- [ ] Multi-objective optimization
  - Pareto frontier 탐색
  - 트레이드오프 시각화 (효능 vs 안전성)

**주요 Deliverable**:
- ✅ RL 최적화 엔진
- ✅ 최적화된 분자 라이브러리 (각 타겟당 1,000개+)

#### Month 5: 가상 스크리닝 및 도킹

  - GLP-1R, GIPR 구조

**Week 3-4: 분자 도킹**

**주요 Deliverable**:
- ✅ 도킹 파이프라인 자동화

### Q4 (10-12월): SaaS 플랫폼 개발 및 출시

#### Month 10-11: 플랫폼 개발

**웹 애플리케이션**
- [ ] Frontend 개발 (React + TypeScript)
  - 분자 그리기 도구 (Ketcher)
  - 3D 분자 뷰어 (3Dmol.js)
  - 결과 시각화 대시보드
- [ ] Backend API (FastAPI + Python)
  - 분자 생성 API
  - ADMET 예측 API
  - 도킹 시뮬레이션 API
  - 최적화 파이프라인 API
- [ ] Azure 배포
  - Container Apps로 마이크로서비스
  - Cosmos DB로 사용자 데이터 관리
  - Blob Storage로 분자 라이브러리 저장

**핵심 기능**
1. **분자 생성 스튜디오**
   - 타겟 조건 입력 (질환, 원하는 특성)
   - AI가 후보 분자 생성 (수백-수천 개)
   - 실시간 스크리닝 및 랭킹
   
2. **가상 스크리닝**
   - 사용자의 화합물 라이브러리 업로드
   - 타겟 단백질에 대한 도킹
   - ADMET 일괄 예측

3. **Lead 최적화**
   - 초기 분자 입력
   - RL 기반 자동 최적화
   - 개선 경로 시각화

4. **프로젝트 관리**
   - 여러 프로젝트 관리
   - 팀 협업 기능
   - 결과 내보내기 (CSV, SDF, PDF)
